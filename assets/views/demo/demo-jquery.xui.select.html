通过将原生控件封装为自定义控件的方式以实现样式自定义， 同时在对自定义控件的任何操作都将同步到原生控件中。

1,xui组件 select
jquery.xui.select.js

Bios Sun



    选择第 个选项

    注意事项

    • 不支持 fieldset 及任何上级元素的 disabled 属性
    • 使用 js 更改所选中项后，必须触发 change 事件以更新 UI
    • 使用 js 增加，删除或修改选项后，必须调用 refresh 操作以更新 UI
    • 不支持多选
    • 不支持 optgrunt 元素
    • 不支持 option 的 disabled 属性
    • 当控件在被设置为 `overflow: hidden;` 的元素中时，其超出部分会被隐藏

    注意事项

    • 不支持 fieldset 及任何上级元素的 disabled 属性

    • 使用 js 修改表单元素的 disabled 属性后，必须调用 refresh 操作以更新 UI
